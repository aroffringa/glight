Glight manual
=============

This is the manual for Glight, a program that turns your computer into a professional DMX controller for live lighting control, with many advanced features.

The source code of Glight can be found here: https://github.com/aroffringa/glight

If you have any questions, feel free to mail me at offringa@gmail.com, or open a ticket
on the github page.
